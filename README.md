# Lucky Draw Registration Form – Ray: Apex Tracksuit  

This is a complete HTML form built for a **Lucky Draw entry** to win the Ray: Apex tracksuit.  

## Features
- Four clear sections:
  1. **Your Account** – name, email, password  
  2. **Shipping & Contact** – phone, address, message  
  3. **Your Order** – choose size, quantity, preferences  
  4. **Finalize Entry** – confirm rules, range sliders, datetime-local  

- Shows different **HTML5 input types**:  
  text, email, password, tel, textarea, select, radio, checkbox, range, datetime-local.
